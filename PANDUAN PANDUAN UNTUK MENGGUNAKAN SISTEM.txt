PANDUAN CARA RUN SISTEM :
clone repositori GitHub ke local
ketikan perintah npm install di terminal yang menagarah ke directorynya untuk install package yang digunakan disistem
setelah berhasil jalankan dengan perintah node app.js untuk menjalankan system nanti otomatis model akan  ke migrate ke database nya , sebelum menjalankan perintah ini buat database toko_app di MySQL , agar nanti semua model akan di migrate menjadi tabel didatabase
jalankan di postman atau thunder client dengan method POST:localhost/:5000/init-produk untuk menambahkan 10produk pada table produk
jalankan node app.js buka di browsere localhost://5000 nanti akan menampilkan card untuk masuk ke sistem admin page, dan chatbot

PANDUAN CARA MENGGUNKAN SISTEM ADMIN PAGE:

buka card sistem admin page
nanti akan ada pilihan input produk, dan cancel pembelian
input produk untuk menambahkan produk ke pembelian
cancel pembelian untuk membatalkan pembelian
